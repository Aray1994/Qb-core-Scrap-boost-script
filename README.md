# Qb-core-Scrap-boost-script
Simple QB-Core Scrapt and boost script!
Adds a scrapyard pad where players can dismantle cars for parts (with configurable rewards and owned‑vehicle protection).
NPC “boost” handler offers steal-and-scrap contracts: a target car spawns at a random location with a map blip and waypoint.
Interactive steps: lockpick → hotwire (police alert option) → automatic repair + keys → GPS reroutes to the nearest scrapyard.
Scrap the boosted car on the pad to finish; rewards scale for contract cars, blips clear automatically.
Cooldowns per player between contracts (configurable).
Fully configurable visuals: scrapyard blips, contract vehicle blips, marker colors, spawn points, vehicle list.
Integrates qb-target, qb-core progress bars/notifications, inventory item boxes, optional police alerts, and force-saves inventory on payout to ensure database persistence.
